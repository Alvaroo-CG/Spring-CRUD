# Spring-CRUD
Java project with CRUD operations done with Spring

## Used technologies

- STS (Spring Tool Suite)
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- MySQL
- Thymeleaf
- Bootstrap
- HTML / CSS
